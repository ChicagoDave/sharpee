What is this?

These are playable demos of interactive fiction games built with Sharpee, a modern TypeScript platform for parser-based IF. Each game runs entirely in your browser — no server required for gameplay.

How to Play

  • Type commands like look, take lamp, go north
  • Examine things: examine mailbox
  • Check inventory: inventory or i
  • Get help: help

About Sharpee

  • Trait-based entity system
  • Language layer separation
  • Capability dispatch for puzzles
  • AI-assisted authoring (coming soon)
