### 2. Plan decisions (author-facing, repo+site, compile-checked)

### 3. VisitedTrait rejected
Existing `RoomTrait.visited` plus design.md §3's "scoring lives on owners, not in trait bodies" already cover the repeated pattern; no new trait needed. One open question parked rather than resolved (see Completed above).
