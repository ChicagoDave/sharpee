## Key Decisions

### 1. Dust edge case resolved by amending ADR-211 Decision 2
The `concealment-test` `dust` fragment (`shelves;{dust}` → `' Thick dust'`/`' Thin dust'`) fit neither the clause (leading comma) nor sentence (leading space + own terminator) category as originally scoped. Resolution: widen the classification character set from `.?!` to `.?!;:` — `;` and `:` already function as the join in English prose, so a comma after them would be wrong (`"shelves;, Thin dust"`). `docs/architecture/adrs/adr-211-description-variety-phrase-system.md` carries a dated amendment note (line ~152) recording this. This is a platform-level ADR amendment, not a story-local workaround.

### 2. Flag day landed as one uncommitted atomic change, by design
Per the plan's flag-day analysis, `resolveSnippetDescription` has no per-story flag — the resolver switch and the full TS migration cannot be split across commits without breaking byte-identity mid-migration. All of Phase 4's sub-steps (resolver, engine gate, all migrations, all test updates) were implemented together and intentionally left uncommitted; this session's finalize is the single atomic commit the plan calls for.
