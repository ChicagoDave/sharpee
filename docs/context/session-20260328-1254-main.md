## Key Decisions

### 1. Bridge Trait Access Pattern
Updated to constructor pattern for consistency with ISSUE-063 changes. Passing the class directly to `entity.get()` rather than the string type key.

### 2. Standalone Walkthrough Shim Approach
Injects all JSON data as a `__DB` global and patches `fetchJson()` to route `/api/*` calls to the in-memory data. The original `index.html` JavaScript is left untouched — only the data injection layer changes. This keeps the build script independent of the viewer's internals.

### 3. ADR-136 Chose Hybrid (Option C)
Pure auto-compute would miss author intent for obscure verbs. Pure annotation would require authors to maintain large lists. Hybrid lets the grammar do the work while giving authors a safety valve. The capability-gated design ensures zero impact on text-only clients.
