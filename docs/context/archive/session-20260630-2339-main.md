### Key finding (belongs in ADR-206)

`params`** object, but `if.event.*` (via `handleGenericEvent`) binds them **flat** (`params = data`).
That inconsistency is the deeper root cause and made `box_rotates` (which renders via `if.event.pushed`)
finicky.
