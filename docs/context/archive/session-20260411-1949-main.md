## Key Decisions

### 1. Document Both Grammar APIs Side by Side
The grammar reference previously showed only one API style. Given that ADR-087 introduced `.forAction()` as the preferred pattern but `.define()` remains correct for phrasal verbs and story-specific commands, both are documented with explicit guidance on when to use each.

### 2. Concealment Commands Added to Grammar Reference
The grammar reference had no entry for the concealment system (ADR-148). Rather than treating it as a gap to file separately, the commands were added during this audit session since all source information was already at hand.
