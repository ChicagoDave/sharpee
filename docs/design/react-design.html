Recommendations

Suggested Default: Literata + Warm Light Theme

Literata was designed by Google specifically for comfortable reading. It has excellent legibility at various sizes and works well for long-form text like IF transcripts.

Transcript Font Literata, 16px
UI Font Inter or system-ui
Command Input JetBrains Mono, 14px

CSS Variables for Author Customization

/* Default theme - authors override these in their story */ :root { /* Typography */ --sharpee-font-transcript: 'Literata', Georgia, serif; --sharpee-font-ui: 'Inter', system-ui, sans-serif; --sharpee-font-input: 'JetBrains Mono', monospace; --sharpee-font-size: 16px; --sharpee-line-height: 1.7; /* Colors - Light theme */ --sharpee-bg: #fffef9; --sharpee-text: #3d3d3d; --sharpee-text-muted: #666666; --sharpee-accent: #6b4423; --sharpee-room-title: #5c4033; --sharpee-command: #6b4423; --sharpee-border: #e0d8c8; /* Spacing */ --sharpee-spacing-sm: 8px; --sharpee-spacing-md: 16px; --sharpee-spacing-lg: 24px; /* Layout */ --sharpee-max-width: 800px; --sharpee-transcript-padding: 20px; } /* Example: Author's dark theme override */ .theme-dark { --sharpee-bg: #1e1e2e; --sharpee-text: #cdd6f4; --sharpee-text-muted: #a6adc8; --sharpee-accent: #89b4fa; --sharpee-room-title: #f5e0dc; --sharpee-command: #89b4fa; --sharpee-border: #313244; } /* Example: Author's retro theme override */ .theme-retro { --sharpee-font-transcript: 'JetBrains Mono', monospace; --sharpee-font-size: 14px; --sharpee-bg: #0a0a0a; --sharpee-text: #00ff00; --sharpee-command: #ffff00; }

How authors use this: Add a <style> block in their story's HTML template, or provide a theme.css file that overrides these variables. The React client applies these variables to all components automatically.

Living Room
This is the living room. There is a door to the east. To the west is a wooden door with strange gothic lettering, which appears to be nailed shut. A large oriental rug covers the center of the room, and a trophy case stands against one wall.
You can see a trophy case (which is closed), an elvish sword, a brass lantern, and an oriental rug here.
> move rug
Moving the rug reveals a trap door.
> open trap door
The door reluctantly opens to reveal a rickety staircase descending into darkness.
> take lantern and sword
brass lantern: Taken.
elvish sword: Taken.
> down
Cellar
This is a dark and damp cellar with a narrow passageway leading east, and a crawlway to the south. To the west is the bottom of a steep metal ramp which is unclimbable.
The door crashes shut, and you hear someone barring it.
