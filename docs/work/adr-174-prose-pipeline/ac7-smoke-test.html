ADR-174 AC-7 — Platform Decoration CSS Smoke Test

This page links the same templates/browser/decorations.css that ships in dist/web/<story>/decorations.css. Each row exercises one entry from the platform vocabulary by emitting the same <span class="sharpee-..."> markup the bracket parser produces for that name.

AC-7 passes if: every row shows the labeled style (italic, bold, color, etc.) without any unstyled "naked" text where a styled span should be.

Switches

[em:emphasized] emphasized expect: italic
[strong:bold] bold expect: bold
[u:underlined] underlined expect: underline
[st:struck] struck expect: line-through
[code:LOOK] LOOK expect: monospace
x[super:2] x2 expect: superscript
H[sub:2]O H2O expect: subscript

IF-semantic classes

[item:brass lantern] You see a brass lantern here.
[npc:the troll] The troll guards the bridge.
[room:West of House] West of House
[direction:north] You go north.
[command:LOOK] Type LOOK to look around.
[quote:said the cat] "You'll regret this," said the cat.

Color palette

[color-red:red text] red text
[color-blue:blue] blue text
[color-green:green] green text
[color-magenta:magenta] magenta text
[bgcolor-yellow:highlight] highlighted

Sizes & fonts

[size-small:fine print] fine print
[size-large:big] big
[font-mono:type-this] type-this

Nesting (AC-3)

[em:[strong:bold italic]] bold italic expect: both
[item:the [em:glowing] lantern] the glowing lantern

Author class (no sharpee- prefix)

[thief-taunt:literal] unstyled (no platform default) expect: plain (story owns CSS)
