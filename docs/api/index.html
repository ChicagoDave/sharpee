Sharpee API Reference

Interactive Fiction engine — world-model & stdlib documentation

Sharpee is a parser-based Interactive Fiction authoring tool built in TypeScript. This reference covers the two core packages that story authors and platform developers interact with most: the world-model (entities, traits, behaviors, spatial relationships) and the stdlib (actions, scope, validation, grammar).
