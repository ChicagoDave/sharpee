This page exercises every shared content primitive through the real MDX
pipeline. It is the pattern every content page follows: a `.tsx` route file
wraps an imported `.mdx` content file in `DocPage`.

## Headings and text

Body text is plain markdown with **bold**, _italic_, and `inline code`.
Links come in two kinds: internal like [the home page](/) (rendered with
`next/link`) and external like [MDX](https://mdxjs.com) (new tab).

### A third-level heading

Lists work as usual:

- Chord is the front door
- The platform section sits behind it
  1. ordered items
  2. nest fine

## Code

Fenced blocks must carry a language tag (`text` at minimum):

```chord
"Fernhill" by Anonymous.

The Orchard is a room. "Rows of apple trees march up the hill."
Tobias is a person in the Orchard, proper.
```
