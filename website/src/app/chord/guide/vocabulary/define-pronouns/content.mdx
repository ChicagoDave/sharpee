For pronouns beyond the standard four sets, you declare a named
set. Each set has exactly five rows (`subject`, `object`,
`possessive`, `possessive-pronoun`, `reflexive`), which you may
list in any order, and you refer to the set by name from a
person's `pronouns` line. Missing or
duplicate rows, shadowing
a standard set, and redefinition are all compile errors.

```chord
define pronouns ze
  subject ze
  object zir
  possessive zir
  possessive-pronoun zirs
  reflexive zirself
end pronouns
```
