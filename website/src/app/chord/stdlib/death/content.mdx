This chapter covers killing the player. Death is not `lose`: `lose`
(chord-language.md) ends the game with an ending event, while the
constructs here run the platform's death machinery: a died event, a
story-visible veto window, and then the defeat ending. chord-language.md
teaches the same constructs from the author's side; this chapter
carries the machinery detail.
