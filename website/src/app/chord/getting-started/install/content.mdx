Chord is a language for writing parser-based interactive fiction. You
describe the world — rooms, things, people, and how they behave — and the
Sharpee platform runs it: in the terminal, in the browser, or under a
transcript test runner.

Everything you need arrives with one command-line tool.

## What you need

- [Node.js](https://nodejs.org) 18 or newer. Check with `node --version`.

## Install the CLI

```shell
npm install -g @sharpee/devkit
```

This installs the `sharpee` command. Run it bare to see what it can do:

```text
sharpee — Interactive Fiction authoring CLI

Usage:
  sharpee build [name|path] [--browser]  Build an author story project (cwd or registered name)
  sharpee init <name>                    Scaffold a new story project
  sharpee compose <file.story> [opts]    Compile a Chord story to Story IR
  sharpee test [name|path] [transcripts…] [--chain] [--stop-on-failure] [--verbose]
                                         Run the project's transcript tests
  sharpee play [name|path]               Play the project interactively (REPL)
```

(The full listing has a few more commands — `register`, `list`, `ifid`,
and the browser-client helpers — you will meet the ones you need as you
go.)

<Callout title="Where the platform comes from">
  A story project declares its own dependency on the Sharpee platform
  packages; `npm install` inside the project fetches them. The global
  `@sharpee/devkit` is just the tool — your story pins the platform
  version it was built against.
</Callout>
